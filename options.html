Permit AI Helper

Enter your OpenAI API key to enable the assistant. The key is stored locally using Chrome sync storage and is only used to call the OpenAI Chat Completions API.

Generate a key at platform.openai.com.

How it works

  • The assistant can summarize form fields and draft polished responses.
  • No data is stored outside of your browser besides requests sent to OpenAI.
  • You can close the sidebar at any time using the × button at the top.
