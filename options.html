Permit AI Helper

Permit AI Helper now talks to CopilotKit. Provide your Copilot Cloud public API key and (optionally) a custom endpoint URL to activate the assistant. Credentials are stored locally via Chrome sync storage and only used for CopilotKit requests.

Create or view keys in Copilot Cloud.

Leave blank to use the default Copilot Cloud endpoint. Provide your own URL if you host CopilotKit yourself.

How it works

  • The assistant can summarize form fields and draft polished responses.
  • No data is stored outside of your browser besides requests sent to CopilotKit.
  • You can close the sidebar at any time using the × button at the top.
